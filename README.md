# Webcam Mirror


This is the quickest way to see yourself. Lightweight and always available.

## To Use

```bash
# Clone this repository
git clone https://github.com/gloomyzerg/webcam-mirror
# Go into the repository
cd webcam-mirror
# Install dependencies
npm install
# Run the app
npm run start
# Package
npm run package
```
